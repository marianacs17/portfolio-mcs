Starbucks interface

Fourth semester, I am learning java, we start a school project. We must do a coffee shop interface. I decide to create mine based on starbucks coffee. So the project consisted of creating an interface for the client to self-serve.
The client was able to make his own order directly in the interface, he was able to choose the coffee he wanted, the kind of milk, the size, the extra ingredients, every customization possible (almost every single one). Of course there were options for size of the coffee, method of payment and if the client was a member of the starbucks rewards he could introduce his card number.
This was a object oriented program on java, it was the first and only semester we took java and I really liked it, the project itself was difficult but fun to do, I enjoy and I'm better at programming visual stuff not really into logical programming.

Arduino project

The next proyect is an Arduino based alarm clock mounted on a Lego chasis we built up.

We created this little car with legos, and using an Arduino mega, a clock, a buzzer, buttons, wires, two little motors and sensors. This was a challenge because it was the first time using Arduino so we had to learn from scratch. We had a lot of help, from teachers, students. Took us a long time due to the learning process, in the meantime we were learning stuff that we were going to implement in the project like the sensors.
